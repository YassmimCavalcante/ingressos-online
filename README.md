# ingressos-online

Ticket Sales Simulator

Um projeto front-end que simula a compra de ingressos para diferentes setores de um evento.

Tecnologias usadas

* HTML5
* CSS3
* JavaScript

Funcionalidades

* Seleção do tipo de ingresso
* Validação de quantidade disponível
* Atualização dinâmica do estoque
* Feedback ao usuário (alertas)

Como funciona

O usuário escolhe:

* Tipo de ingresso (pista, superior ou inferior)
* Quantidade desejada

O sistema:

* Verifica se há disponibilidade
* Atualiza o número de ingressos restantes
* Exibe mensagens de sucesso ou erro

Aprendizados

* Condicionais (`if/else`)
* Manipulação do DOM
* Conversão de dados (`parseInt`)
* Controle de estado
